选博客模板时，外观只是第一层。更重要的是它会不会影响以后写东西。

## 几个方向

Astro 很适合内容站，轻、快、写作体验清爽。Hugo 的生成速度非常快，主题生态也成熟。Jekyll 和 GitHub Pages 绑定很自然，适合技术写作。Hexo 在中文博客圈很常见，主题丰富，迁移资料也多。

最后选择 Next.js，是因为它给未来留下了更多余地。

## 这版博客的原则

- 文章优先，页面不要抢内容的风头
- 首页直接进入博客体验
- 样式克制，但不要冷冰冰
- 内容结构要能被 Git 长期管理

## 关于模板

这不是直接套某一个模板，而是把个人博客需要的核心部分重新整理成一个干净的 Next.js 项目。这样后面换视觉风格、加功能或迁移部署平台都会更稳。
