# Learning Notes

This section stores reusable academic notes. A good note should be clear enough to revisit months later.

这里适合放概念解释、公式、代码片段、工具比较、课程整理和博士准备材料。

## Areas

- Bioinformatics: [[bioinformatics-sequence-quality-control]]
- Statistics: [[statistics-linear-model-thinking]]
- Coding: [[coding-reproducible-python-workflow]]
- PhD preparation: reading plans, research interests, writing practice

## Note standard

Each note should include:

- What problem this concept helps solve
- Key assumptions
- Minimal example or equation
- Links to related notes
- Open questions
