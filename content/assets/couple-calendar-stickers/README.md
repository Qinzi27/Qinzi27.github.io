# Couple Calendar Stickers

Put public-safe decorative stickers for `content/Our Calendar/index.md` here.

Supported formats:

- `.png`
- `.jpg`
- `.jpeg`
- `.webp`
- `.gif`
- `.svg`
- `.avif`

Run `npm run generate-calendar-stickers`, `npm run preview`, or `npm run build` to refresh the sticker area.

Important: files in this folder are copied as public site assets. Do not place private photos, screenshots, or sensitive images here.
