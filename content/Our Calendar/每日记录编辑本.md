# 每日记录编辑本

这个文件是以后主要编辑的地方。保存后运行 `npm.cmd run preview` 或 `npm.cmd run build`，系统会自动同步到加密页面 `content/Our Calendar/index.md`。

## 怎么新增睡眠记录

复制下面这个模板，把日期和时间改掉即可。页面里会优先显示“宝宝几点睡着”，不会显示聊天昵称。

```md
## 2026-06-30
sleep: 01:20左右

### 碎碎念
- 这里可以放一点当天想留下的小话。
```

如果是第二天补记，比如 `2026-06-30 上午 10:00` 写“昨晚一点睡着”，日期请写成 `2026-06-29`。
